# Documentation

This directory contains documentation and sample data files.

## university-handbook-sample.txt

A sample university financial aid handbook with policies including:
- Tuition payment policies
- Hardship extension procedures
- Emergency grant programs
- Financial aid appeals
- Bursar's office policies

This file is used for testing and can be uploaded to Azure AI Search using the `scripts/upload_handbook.py` script.

**Note:** Replace this with your actual university handbook for production use.
